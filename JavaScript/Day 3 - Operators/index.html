Operators

  • Arithmetic Operators: +, -, *, /, %, **, ++, --
  • Assignment Operators: =, +=, -=, *=, /=, %=, **=, <<=, >>=, >>>=, &=, ^=, |=
  • Comparison Operators: ==, !=, ===, !==, >, <, >=, <=
  • Logical Operators: &&, ||, !
  • Bitwise Operators: &, |, ^, ~, <<, >>, >>>
  • String Operators: +, +=
  • Conditional (Ternary) Operator: ? :
  • Type Operators: typeof, instanceof, delete
