Hello JavaScript!

  • 1995: JavaScript was created by Brendan Eich while he was working at Netscape Communications
  • 1995: JavaScript was initially called "Mocha", but was later renamed to JavaScript
  • 1996: JavaScript 1.1 was released, adding support for arrays and other features
  • 1997: ECMAScript 1 was released, standardizing the language and paving the way for future development
  • 1998: JavaScript 1.2 was released, adding support for regular expressions and other features
  • 1999: ECMAScript 3 was released, adding support for try-catch blocks and other features
  • 2000s: JavaScript began to gain popularity as a server-side language, with the rise of Node.js and other frameworks
  • 2009: ECMAScript 5 was released, adding support for strict mode and other features
  • 2015: ECMAScript 6 was released, adding support for classes, promises, and other features
  • Present day: JavaScript continues to evolve, with new features and updates being added regularly
